May 2024

Company Structure

The Organizational Hierarchy Management System is a programmatic solution designed to model and manage employee structures within a corporate environment. By implementing a binary tree data structure, the system organizes employees hierarchically, starting with a CEO and enabling dynamic addition of subordinates. It supports operations like searching, sorting, and analyzing employee data based on attributes such as job title, age, years served, and employee ID. Through interactive command-line menus, users can efficiently navigate roles, update hierarchies, and generate sorted lists of employees. The project emphasizes efficient traversal algorithms and string manipulation to streamline organizational operations, offering insights into workforce management and hierarchical scalability.

Banking System APP

The Banking System Project is a user-friendly application designed to manage virtual bank accounts efficiently. It provides core banking functionalities such as user registration, deposits, withdrawals, balance inquiries, and account deletion through an interactive GUI built with customtkinter. By storing user data in a JSON file, the system ensures persistence and secure record-keeping. The application allows users to interact seamlessly via a modern interface while leveraging data-driven operations to maintain accurate account balances and transaction history.
